# Emotional Design 情感設計探討

## 前言
在當今競爭激烈的市場中，產品設計不僅需要滿足功能需求，還需要觸及用戶的情感層面。產品設計師不是單純只有做出一個「好用」的產品給使用者使用。可以發現情感層面的重要性已經是產品需要具有的層面，就像是需求金字塔現階段已經有許多產品能夠滿足基本的功能需求，而要比這些基本滿足功能需求的產品更好，需要思考的層面也就更多。

而情感設計(Emotional Design)是以提升用戶體驗為目標，透過喚起情感共鳴來促進產品與用戶的深層連結。包含智慧型手機的手感設計到友善的介面動畫，在大多數產品都能滿足功能需求的狀況下，情感設計已成為創造獨特且有吸引力的產品的關鍵市場策略。

## 什麼是情感設計?
情感設計(Emotional Design)是產品設計中的一個重要理念，主要的目的為滿足用戶的情感需求，提升使用者整體產品使用體驗。情感設計的概念是由 Don Norman 在其著作《Emotional Design》中首次系統化提出，Don Norman 認為設計不僅需要實用性以使用者為中心的設計思考之外，還需要觸發用戶的情感共鳴。

在《Emotional Design》這本書中，Don Norman 將情感設計劃分為三個層面，每個層面說明不同的心理層面影響用戶的感受與行為。分別為本能層面(Visceral Level)、行為層面(Behavioral Level)、反思層面(Reflective Level)。

#### 本能層面(Visceral Level)
本能層面主要為專注於產品的感官屬性，產品的造型外觀、觸感和材質等。使用者第一眼對於產品的第一印象就是本能層面。
本能層面與使用者的潛在意識相關，會影響使用者來決定是否值得擁有該產品。

行為層面關注的是產品的功能性與可用性，最直接影響行為層面的是使用者的操作感受，產品除了滿足實際需求之外，關鍵在於使用者在產品交互體驗的過程中感到流暢、舒適。

#### 反思層面(Reflective Level)
反思層面涉及的是使用者對產品的深層感受與自我投射，強調產品的故事性和意義。反思層面建立的是產品與其使用者的長期情感連結，影響的是使用者對其品牌忠誠度。

### 為什麼情感設計越來越重要?
差異在於含有情感設計的產品可以滿足使用者的心理需求，而主要原因被逐漸重視的原因在於現代產品的市場競爭十分激烈，只單純具備功能的產品已經不足以吸引用戶，設計除了需要基本的功能需求外，還須滿足更深層次的心理需求。

以UX的角度下檢視，情感設計與用戶體驗設計是密不可分的，情感設計是用戶體驗中的針對情感部分的延伸，情感設計的核心在於理解用戶的情感旅程，並設計出除了能夠解決問題之外還能帶給使用者心理滿足的產品。

## 情感設計核心原則
情感設計的核心原則在於需先滿足用戶需求，可以說滿足用戶需求是金字塔的底部，一個連用戶需求都滿足不了的產品是沒辦法帶給使用者良好體驗的，更何況要與使用者建立情感連結。

在滿足用戶需求之後，需要創造良好的體驗互動，觸發使用者的正面情緒、減少用戶的負面情緒，讓用戶對於產品有良好的記憶點。透過良好的記憶點建立情感連結，品牌需要有相關的故事性或品牌文化，增加其受眾對於品牌的認同感。

### 滿足用戶需求
在產品使用流程中，能夠創造愉快的互動過程，讓使用者除了解決問題之外還感到滿足。
> [!Tip] 
> Spotify 提供用戶播放聆聽音樂功能之外，還提供個性化播放清單，讓用戶感受到被重視且顯示其對每個用戶的獨一無二。

## 如何將情感設計融入產品?
產品如果要將情感設計融入產品，需要從用戶心理、產品功能和品牌價值多層面出發，並通過系統性的設計策略來喚起使用者的正向情感反應。

情感設計的核心在於更加了解使用者的內在動機、情感期待和痛點，因此首要的是進行用戶研究。觀察用戶後需建立正向的互動體驗，讓用戶在與產品的每一次互動中都感到愉快和滿足，包含產品的外觀、色彩搭配和語言風格，都是情感設計的重要載體。
需要設計引發情感共鳴的視覺與語言，最後需要結合品牌價值，並且賦予產品意義。

情感設計並非一次性投入，而是產品設計中持續的核心策略。只有透過深入了解使用者、優化互動細節、融入品牌價值以及持續測試迭代設計，才能成功地將情感設計轉化為使用者的滿意度與忠誠度。

> [!Note] 
> 以 Nintendo Switch 為例:

> 研究情感需求：了解到玩家希望獲得沉浸式遊戲體驗，又需要靈活性。

> 設計正向互動：加入手把動態振動功能，讓玩家在遊戲過程中有更多沉浸感與遊戲體驗。

> 引發共鳴的設計：外觀簡約、便攜且可不只單人遊玩，適合年輕用戶揪好友同樂的生活方式。

> 結合品牌價值：透過任天堂的家用遊戲機文化，讓使用者感受到共享的快樂。

### Nike Run Club App
Nike 本身在品牌一向提倡「Just Do It」的精神，Nike Run Club App也是透過情感設計將品牌價值深植於使用者心理，讓跑步不僅僅是運動，更是一種自我挑戰的象徵。

讓冷冰冰的運動數據轉化為有故事的體驗，Nike Run Club App除了有設置挑戰之外，還記錄使用者的進步曲線，使用者可以在App中見證自己一天一天的成長。

#### 本能層面(Visceral Level)
在產品的視覺設計上，整體使用介面充滿活力的色彩(高明視度)搭配(如螢光綠、亮橙色、深藍色)和簡潔的版面配置設計，主畫面強調即刻運動的力量與激情，點擊按鈕立即激勵使用者開始跑步。
微互動的動態效果加入: 當使用者完成其設定目標後，動畫或數據的顯示方式增強了視覺刺激，完成成就獎章顯示讓人產生滿足感。

#### 行為層面(Behavioral Level)
產品在實用性與易用性，在其功能設計專注於記錄跑步距離、速度、時間等，並且提供個性化建議，讓使用者可以清楚了解自己進步的軌跡。

提供語音教練能夠即時語音提示結合激勵語句，不僅指導使用者更高效訓練，也增強了陪伴感和支持感。

#### 反思層面(Reflective Level)
Nike Run Club App含有專屬獎章與挑戰目標，當使用者完成特定挑戰時，會展示獨一無二的數位獎章，使用者可以真切感受到自己的努力被認可，並激發繼續挑戰的動力。

Nike Run Club App還有強大的社群功能，使用者可以與朋友分享運動成果或參加全球挑戰，強化了社交認同與品牌忠誠。

## 總結
在競爭激烈的產品市場中，功能性不再是唯一的制勝關鍵，情感設計成為提升產品吸引力與建立品牌忠誠的利器。透過滿足用戶的心理需求，創造良好的互動體驗，並賦予產品品牌價值與故事性，情感設計將冷冰冰的功能轉化為溫暖的人性化體驗。

無論是 Nike Run Club App 激發用戶運動激情，還是 Nintendo Switch 喚起情感共鳴，都展示了情感設計如何讓產品成為使用者生活的一部分。這不僅提升了使用滿意度，更讓品牌在用戶心中留下了深刻印象。

情感設計的本質在於超越實用性，為產品注入「情感的溫度」。也正是提醒我們，設計的真正價值在於與人產生連結，並讓使用者從每一次互動中感受到快樂與滿足。

> [!IMPORTANT]  
> 情感，讓產品從眾多競品中脫穎而出，成為不可或缺的一部分。。
