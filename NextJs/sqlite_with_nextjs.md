# Using SQLite with Next.js
## 安裝 Prisma
`Prisma` 是一套可以透過 `JavaScript` 或 `TypeScript` 來操作資料庫的工具，銜接 PostgreSQL、MySQL、SQLite、MongoDB 的資料庫，並提供 GUI 介面。

透過 npm 安裝 Prisma

```shell
npm i -D prisma
```

安裝 Prisma cilent
```shell
npm i @prisma/client
```

## 設置 schema.prisma 
初始化 Prisma
```shell
npx prisma init
```
